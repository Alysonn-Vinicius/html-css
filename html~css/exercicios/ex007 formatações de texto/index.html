PRINCIPAIS FORMATAÇÕES

Texto Sublinhado

Usando a tag ins texto sublinhado fica assim mas, também podemos usar a tag "U" que "fica assim" porém a tag "U" é uma não semantica.

Texto em Negrito

Usamos a tag STRONG para deixar em negrito e com significado ou seja, semantico.
usamos a tag <b> e fica assim texto em negrito(porém assim é a tag não semantico)

Texto em Ênfase/Itálico

Usamos a tag "em" para deixar em itálico e com significado
Usamos a tag "i" para deixar o texto em itálico.

Texto Deletado

Usando a tag del o texto ficaria assim

Texto Sobrescrito

Usado a tag sub para deixar com letras acima, exemplo: x2+3

Texto Subscrito

Usamos para colocar a letra abaixo da linha, exemplo: H2O

Texto Marcado

Usamos a tag mark para deixar o texto como se fosse um marcador
