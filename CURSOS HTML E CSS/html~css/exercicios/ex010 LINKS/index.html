Links

Links externos

Usando a tag <a href=""> podemos colocar o link externo(encaminha para outro site.) usamos os parâmetros target="_blank" rel="external" que diz para o navegador que é um link externo e que é para abrir numa nova guia.

EX: esse é meu instagram pessoal clique aqui

Links Internos

É como se fosse uma segunda aba do site, sendo encaminhado pega mesma tag porém mudando o "rel" de external para "next"

ex: clique aqui para a proxima página

Link para Download

Usamos a mesma tag <a href="" domwload="" type"/"> o href é o link do download, o "download" vai com o nome e arquivo, o "type" é o media types dos arquivos indicando para o navegador que o tipo de arquivo que será baixado.
