DIsplay

Usamos o DIsplay para definir tipo.

  • Inline: são tags que alinham os elementos todos numa mesma linha e a borda só servem até o conteudo. ex(span,a,img)
  • block: são tags que alinham os elementos todos um a baixo do outro , ocupando todo uma linha. ex(h1, p, table)
  • Inline-block: Sua largura é definida com base na sua width. e alinham os elementos um abaixo do outro.
