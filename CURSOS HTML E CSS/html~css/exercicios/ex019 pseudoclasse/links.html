Links Personalizado:

Com as Pseudoclasses, podemos alternar a cor e formato de decoração dos links.

Exemplo:

Personalizando links

  • Meu Instagram
  • Meu GitHub
  • Meu facebook

Por padrão os links, ainda não abertos serão na cor azul e sublinhado e os já abertos, ficam na cor margenta(roxeado); Podemos fazer uma alteração nas duas formas com a Pseudoclasse ;visited
