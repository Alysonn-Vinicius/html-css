font-family

é usada sempre mais de uma fonte para que hajá compatibilidade do navegador ao dispositivo, tendo categorias como:

  • serifadas: são fontes com serifas que ajudam e facilitam na leitura, usada principalmente em titulos.
  • sans-serif: fontes sem serifas, recomendado para textos longos em sites.
  • monoespaçadas: sem serifas e ocupa um espaço maior tendo um distaciamento entre as glifos(letras)
  • handwriting: font que simula a escrita a mão
  • display ou comemorativas: não se preocupa com nenhuma das regras anteriores

font-size

é usada para o tamanho da fonte do seu texto no site, tendo medidas relatias(em, ex, rem, vh, vw, %) e absolutas(cm, mm, in, pt, px, pc) normalmente o usado nos navegadores é "16px" ou em "em" que é geralmente usado. obs: 16px geralmente equivale a 1em
