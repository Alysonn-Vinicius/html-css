Exemplo de cores/ "NOMES"

É o meio mais padrão, digitando o nome da cor em inglês, sendo muito limitado o tanto de cores disponiveis.

Exemplo de cores/ "HEXADECIMAIS"

Neste parágrafo, mudamos a cor por meio de códigos hexadecimais, usando apenas os valores do css e usa a "#" antes do código em hexadecimais.

Exemplo de cores/ "RGB()"

Nste parágrafo, mudamos as cores pelo valor "rgb(R,G,B) indicando o valor para cada um das cores sendo o máximo para cada o valor 255

Exemplo de cores/ "HSL()"

Neste, mudamos por meio do valor "hsl(0,0%,100%)" sendo os dois ultimos escolhidos pelo uso de porcentagem.

Exemplo de cores/ "MOUSE POR CIMA"

Esse é o meio mais pratico e que particulamente eu mais vou usar, passando o mouse por cima da cor e escolhendo a cor de preferência, podendo mudar até a transpârencia.
