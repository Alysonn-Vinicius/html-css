Continuações de formatações

codigo de computador

Usando a tag <code> mostra que é um código de computador para o navegador e tem bastante valor semantico. ex: num = int(input('digite um número')) if num % 2 == 0: print(f'0 numero {num} é par') else: print(f'0 número{num} é impar') print('FIM DO PROGRAMA')

Código de computador usando o pre

Usando a tag "pre" antes da tag "code" para usar digitar um codigo irá mostrar extamente como foi escrito! 


            num = int(input('digite um número'))
            if num % 2 == 0:
                print(f'0 numero {num} é par')
            else:
                print(f'0 número{num} é impar')
            print('FIM DO PROGRAMA')
