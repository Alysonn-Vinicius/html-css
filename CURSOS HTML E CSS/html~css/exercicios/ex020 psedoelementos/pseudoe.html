::BEFORE

podemos colocar coisas antes de um elemento html, escrevendo apenas no css e dizendo onde ele irá ficar através da class.

EX: 

        
            <style>
          a::after{
              content: '=>';    
              </style>

::AFTER

podemos colocar coisas depois de um elemento html, escrevendo apenas no css e dizendo onde ele irá ficar através da class.

EX: 

        
            <style>
          a::after{
              content: '=>';    
              </style>
