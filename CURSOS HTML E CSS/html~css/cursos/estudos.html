REDES DE COMPUTADORES

O que é?

É a conexão entre os dispositivos interligados.

Classificação de Redes:

As redes podem conter 3 classifações sendo elas: "LAN, MAN E WAN".
LAN = local area network, é uma rede pequena como uma casa, empresa ou loja. MAN = Metropolitan area network, é uma rede de médio porte podendo ser contém uma área mais abragente como uma cidade. WAN = wide area nework, é uma area ainda maior, podendo ser um país ou continente.
