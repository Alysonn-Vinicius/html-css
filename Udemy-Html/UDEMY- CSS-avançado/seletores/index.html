Tipos de Seletores

Seletores Universais

Aplica-se em todos os elementos no documento.
Ex: *{}Seleciona todos os elementos da página.

Seletores de texto

É o mais usado comumente, seleciona o pela tag e as configurações são aplicadas em todos as tgas.
Ex: h1{} vai marcar todos os h1 da página.

Seletores de Classe

Seleciona um elemento cujo atributo class tem o valor especificado depois do ponto.
Ex: .verde{"algum stilo"} sendo aplicados a todos que tem o atributo class com o valor verde.
Ex2: p.verde{} será aplicado somente nas tags "p" que contenha o atributo class.

Seletores de Id

Seleciona um elemento cujo atributo ID tem o valor especificado após o símbolo de #.
Ex: #cabecalho{"algum stilo"} sendo aplicados a todos que tem o atributo ID com o valor cabecalho.

Seletor de filho

Seleciona um elemento que é filho direto de outro.
Ex:li > a {} Seleciona quaisquer elementos "a" que são filhos de um elemento "li" (mas não outros elementos a na página).
Ex2:

  • Home
  • sobre

    • Não é filho direto como está dentro de outra tag mesmo perto, não é filho direto e não receberá a configuração

    Seletor de descendente

    Seleciona um elemento que é descendente de outro elemento especificado (e não apenas filho direto desse elemento).
    Ex: p a {}Seleciona quaisquer elementos "a" que residem dentro de um elemento <"p"> mesmo e houver outros elementos aninhados entre ele.
    Diferente do seletor filho, ele pega todos os elemntos que estão dentro da primeira tag.

    Seletor de irmão adjacente

    Seleciona um elemento que é o irmãopróximo de outro.
    Ex:h1+p {} Seleciona o primeiro elemento <"p"> depois de qualquer elemento <"h1">(mas não outros elementos <"p">).

    Seletor de irmão geral

    Seleciona um elemento que é um irmão de outro, embora ele não precise ser o elemento diretamente precedente sendo todos que forem irmãos seria aplicado, ao contrário do outro que seria somente o primerio.
    Ex: h1~p {}Se houvesse dois elementos <"p"> que fossem irmãos de um elemento <"h1">, essa regra se aplicaria ao dois.
