TRABALHANDO COM LISTAS

Listas Ordenadas

são listas que contem ordem, podemos criar usando a tag <ol> e seus itens tem que conter a tag <li> que tem o fechamento opcional
A tag ol tem um parametro "type" como :

  1. 1 - valor padrão, cria listas numeradas. ex: 1,2,3,4,5...
  2. A - cria listas alfabeticas em maisculas. ex: A,B,C,D,E...
  3. a - cria listas alfabeticas em minusculas. ex: a,b,c,d,e....
  4. I - cria listas em algarismo romanos maisculas. ex: I,II,III,IV,V...
  5. i - cria listas em algarismo romanos minusculas. ex: i,ii,iii,iv,v...

ela também tem um parâmetro chamado "start" que cria uma contagem e segue até o número indicado ex: "<ol type="i" start="2">"

Lista De Definições

usamos a tag <dl> para meio que criar uma lista de Definições, vem acompanhada das tag "dt" e "dd". O termo é mencionado pela tag <dt> e seu significado é inscrito pela tag <dd> como por exemplo:

HTML [usando a tag "dt" por ser o titulo]
é uma linguagem de marcação de texto, usada para criar conteúdo de site.[usando tag da "dd" que é definição de algo]
